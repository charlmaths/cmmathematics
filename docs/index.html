I'm a mathematician learning how to build things!

Introduction

I'm a Junior Data Engineer based in UK. I have a mathematics degree and spend my working days learning and building data pipelines on GCP for a financial institution. You know, the kind of work that teaches you a lot about systems you didn't design! (actually cracked way to learn tbh). Outside work I'm constantly trying to increase my D.E. fundamentals

Outside of that, I'm teaching myself to build mathematical simulations in Python and MATLAB. Differential equations, orbital mechanics, signal processing. The kind of problems where you define the physics as equations, write code to solve them, and plot the result to see if the universe agrees with you.

The long-term goal is algorithms and simulation engineering in the space sector or adjacent to it. This site is the working notebook for getting there — field notes on problems I've solved, things I've built, and concepts I'm still figuring out. It is not polished. It is honest.

Oh, I also want to dabble with robotics and electronics
